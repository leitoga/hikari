# Hikari · Lámparas 3D

Sitio web estático de Hikari — catálogo de lámparas artesanales.

## Contenido
- `index.html` — la página (abrila en cualquier navegador)
- `support.js` — runtime necesario (no borrar)
- `fotos/` — las 22 fotos del catálogo (Lampara-01 … Lampara-22)

## Cómo publicarlo en GitHub Pages
1. Subí **todo el contenido de esta carpeta** a un repositorio (manteniendo la estructura: `index.html`, `support.js` y la carpeta `fotos/` juntos).
2. En el repo: **Settings → Pages**.
3. En *Source* elegí la rama (`main`) y la carpeta `/ (root)`. Guardá.
4. A los pocos minutos tu sitio queda online en `https://TU-USUARIO.github.io/TU-REPO/`.
